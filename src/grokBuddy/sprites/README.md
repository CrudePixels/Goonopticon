# Grok Buddy sprites

Replace these placeholder images with your own. Recommended size: **88×100** (or same aspect) so they match the hex container.

| File | Use |
|------|-----|
| `idle_1.png` | Idle animation frame 1 |
| `idle_2.png` | Idle animation frame 2 |
| `walk_1.png` | Walk animation frame 1 |
| `walk_2.png` | Walk animation frame 2 |
| `alert.png` | Alert / goon / lolcow popup |
| `investigate.png` | Fake investigation mode |

If a file is missing or fails to load, the app falls back to the CSS hex "GROK" shape.
