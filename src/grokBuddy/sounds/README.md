# Grok Buddy sounds

Replace these placeholders with your own. Format: WAV (or MP3; adjust code if needed).

| File | Use |
|------|-----|
| `pop.wav` | When the speech bubble appears |
| `alert.wav` | Goon / lolcow / alert lines |
| `hmm.wav` | When Grok is "thinking" or idle comment |

Keep clips short (under ~1s). If a file is missing, playback is skipped.
