# Custom chat emotes

Add or replace `.png` files here. Use the filename (without `.png`) as the shortcode in chat.

**Example:** `kappapride.png` → type `:kappapride:` in a message to show the emote.

- Only `.png` files are used.
- Shortcodes are case-sensitive and must be wrapped in colons: `:name:`.
- Keep images small (e.g. 28×28 or 56×56) for readable chat.
