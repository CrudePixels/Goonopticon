# User-customizable content

Edit files here to personalize the app. Changes take effect on next launch (or after restarting the app).

- **splash/** — `quotes.txt`: one line per quote (random in footer). `loading.txt`: one line per loading message (cycles on splash screen). Lines starting with `#` are comments.
- **themes/** — JSON theme files. Add a new `.json` (e.g. `mytheme.json`) and add an option in Settings to use it.
- **icons/** — App icon (`icon.ico` / `icon.png`) and tray icon (`tray.png`). See `icons/README.md`.
- **sprites/** — Sprite sheets and UI image assets. See `sprites/README.md`.
- **animations/** — Animation assets and CSS/keyframes. See `animations/README.md`.
