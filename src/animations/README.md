# Animations

Place animation assets and definitions here.

- **CSS** — Keyframe or transition definitions (e.g. `animations.css`) that the renderer can link or import.
- **Sprite animations** — Frame sequences as separate PNGs or sprite sheets; drive with JS or CSS `animation` + `steps()`.
- **Lottie / JSON** — If you add a Lottie player, put `.json` animation files here.
- **Other** — Any motion or transition assets used by the app (loading indicators, scanline effects, etc.).

Reference from the renderer by path or import. For CSS, add a link in your HTML to `../src/animations/animations.css` (or copy into renderer styles).
