# PodAwful's Timestamps

A browser extension for timestamping videos with group, tag, and note management.

## Development

- Run `npm install`
- Build with `npm run build`
- Edit code in `JS/`, styles in `CSS/`
- See `tests/` for unit tests (to be added)

## Features

- Sidebar with groups, notes, tags
- Theme support
- Hotkeys
- Tag manager
